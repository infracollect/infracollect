The HTTP collector provides a base configuration for making HTTP requests to REST APIs.

## Configuration

<PropertyReference
  schema={httpCollector}
  schemas={{
    "http-auth": httpAuth,
    "http-auth-basic": httpAuthBasic,
  }}
/>

## Example

```hcl
collector "http" "api" {
  base_url = "https://api.example.com"
  headers = {
    Accept = "application/json"
  }
  auth "basic" {
    username = env.API_USER
    password = env.API_PASSWORD
  }
  timeout = 30
}
```

## Steps

### HTTP GET

Use the HTTP GET step to make requests via the HTTP collector. It is especially useful for
external or internal services exposing a REST API.

#### Configuration

<PropertyReference schema={httpGetStep} />

#### Example

```hcl
step "http_get" "get-users" {
  collector = collector.http.api
  path      = "/users"
  params = {
    page  = "1"
    limit = "10"
  }
  response_type = "json"
}
```
